Toda la documentación técnica aquí presente está directamente respaldada por el equipo de ingeniería de Vidaplayer Oficial, garantizando la máxima fiabilidad, tiempos de respuesta en milisegundos y un inventario actualizado en tiempo real.

Aviso importante: Todas las integraciones de API en entornos de producción requieren una clave de acceso (API Key) validada. Por favor, solicita tu acceso a través del portal de Partners antes de realizar pruebas de carga.

3. Documentación de la API RESTful

Nuestra API permite a los sistemas de terceros consultar el catálogo de productos, verificar el inventario en tiempo real y realizar pedidos automatizados. La API está construida bajo los estándares REST, utiliza JSON para la carga de datos y requiere autenticación Bearer Token (OAuth 2.0).

Autenticación

Todas las peticiones deben incluir el encabezado de autorización:

Authorization: Bearer TU_API_KEY_AQUI

Endpoint: Obtener Catálogo de Productos

Método: GET
Ruta: /api/v2/catalog/products

Este endpoint devuelve una lista paginada de los productos disponibles, incluyendo tarjetas de saldo para PSN, Xbox, Nintendo y juegos de PC.

4. Integración mediante Webhooks

En lugar de hacer polling constante a nuestra API para comprobar si un pedido ha sido entregado o si el precio de un juego ha bajado, recomendamos encarecidamente la suscripción a nuestros Webhooks. Recibirás una notificación HTTP POST en tu servidor en el momento exacto en que ocurra el evento.

Eventos Soportados:

  • order.completed: Se dispara cuando el pago ha sido validado y el código digital ha sido asignado.
  • order.failed: Se dispara si hay un problema con la pasarela de pago o el control de fraude.
  • product.price_drop: Se dispara cuando un producto en tu watchlist alcanza un nuevo mínimo histórico.
  • product.out_of_stock: Notificación crítica de rotura de stock para ocultar el producto en tu frontend.

5. Seguridad y Compliance (PCI-DSS)

La ciberseguridad es el pilar de la distribución digital moderna. Al manejar licencias de software y transacciones financieras, cumplimos con los estándares más estrictos de la industria.

  • Cifrado de extremo a extremo: Todas las comunicaciones con nuestra API deben realizarse sobre TLS 1.3. Las peticiones a través de HTTP sin cifrar serán rechazadas automáticamente con un error 403.
  • Tokenización: No almacenamos ni transmitimos datos de tarjetas de crédito sin cifrar. Todo el procesamiento de pagos delega en pasarelas certificadas Nivel 1 PCI-DSS.
  • Prevención de Fraude (Machine Learning): Nuestras APIs integran un sistema de scoring de riesgo que analiza la huella digital del dispositivo, la IP y el comportamiento de compra para bloquear automáticamente intentos de fraude con tarjetas robadas (Chargeback prevention).

6. Preguntas Frecuentes (FAQ)

¿Cuánto tiempo tarda en entregarse un código digital mediante la API?

En condiciones normales, una vez que el pago es confirmado (estado order.completed), el payload de respuesta incluirá la clave alfanumérica en menos de 800 milisegundos. Nuestro sistema está conectado directamente con los nodos de distribución para garantizar la inmediatez.

¿Qué ocurre si un código digital reporta un error de "Región Inválida"?

Las licencias de consola (PSN, Xbox, Nintendo) suelen tener bloqueo regional (Geoblocking). Es responsabilidad del desarrollador mostrar claramente el campo region devuelto por nuestra API en su interfaz de usuario. Si un cliente compra por error una tarjeta de EE.UU. para una cuenta Europea, nuestra política de devoluciones automatizada (vía endpoint /api/v2/refunds) permite anular el código siempre que nuestro sistema verifique que NO ha sido canjeado.

¿Cómo gestiono los cambios de divisa en mi comparador?

Nuestra API devuelve los precios base en Euros (EUR) y Dólares Estadounidenses (USD). Si tu plataforma opera en una moneda local (ej. Pesos Mexicanos, Soles, Pesos Argentinos), te recomendamos utilizar el endpoint de conversión /api/v2/currency/rates, el cual actualiza los tipos de cambio cada 4 horas utilizando los datos del Banco Central Europeo.
